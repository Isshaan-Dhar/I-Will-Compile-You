# Custom Compiler & Stack-Based VM
A modular language processor that demonstrates the Analysis and Synthesis phases of Compiler Design.

### Compilation Pipeline
1. **Lexical Analysis:** Tokenizes raw input using Regular Expressions.
2. **Parsing:** Constructs an Abstract Syntax Tree (AST) to define program logic.
3. **Synthesis:** Compiles the AST into custom stack-based Bytecode.
4. **Execution:** Runs instructions on a custom Virtual Machine (VM).
